# 2025 Lab Project Topics

## Overview

**General Theme**: Collaborative entertainment applications

The entertainment value should come from multiple users participating together in shared activities. Focus on real-time interaction and social engagement.

## Project Ideas

### 1. Live Collaborative Playlist Builder / Music Jam Session
Create a platform where multiple people can contribute songs to a shared playlist. Features could include:
- Real-time song suggestions and additions
- Dynamic voting system to reorder or remove tracks
- Live listening sessions with synchronized playback
- User preferences and recommendation algorithms

### 2. Live Trivia Game
Develop a real-time trivia platform with automated gameplay. Features could include:
- Timer-based trivia questions with automatic progression
- Real-time participation and scoring
- Ranking systems and leaderboards
- Tournament brackets based on user performance
- Multiple difficulty levels and categories

### 3. Mini Arcade Tournament
Build a collection of simple multiplayer games in a tournament format. Features could include:
- Classic games like tic-tac-toe, connect-4, or similar
- Gauntlet-style elimination tournaments
- Real-time matchmaking
- Progressive difficulty and skill-based pairing
- Tournament brackets and winner tracking

### 4. Photo Scavenger Hunt
Create a collaborative photo challenge platform. Features could include:
- Time-limited photo challenges with specific themes/tasks
- Photo upload and sharing functionality
- Community voting system for submissions
- Real-time leaderboards and scoring
- Social features for sharing and commenting
